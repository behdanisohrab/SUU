
# Simple Ubuntu Utility

SUU or Simple Ubuntu Utility is a bash script that designed to make Ubuntu more useful by removing snaps and do more.

For run:
'git clone https://github.com/behdaisohrab/SUU;cd SUU;./main.sh'
